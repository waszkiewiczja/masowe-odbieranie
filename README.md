# Email Multi-Account App

Aplikacja do odbierania i wyświetlania wiadomości e-mail z wielu skrzynek pocztowych (POP).

## Backend (Node.js)

1. Przejdź do folderu `backend`:
   ```sh
   cd backend
   ```
2. Uruchom backend:
   ```sh
   npm start
   ```
3. Backend będzie dostępny na `http://localhost:3001/api/emails`.

## Frontend (React)

1. Przejdź do folderu `frontend`:
   ```sh
   cd frontend
   ```
2. Uruchom frontend:
   ```sh
   npm start
   ```
3. Frontend będzie dostępny na `http://localhost:3000`.

## Uwaga

- Backend i frontend muszą być uruchomione jednocześnie.

- Hasła i dane dostępowe przechowywane są w pliku `config.json` (niezalecane w produkcji).
